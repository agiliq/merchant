----------------
Django-Merchant
----------------

|TravisCI|_

.. |TravisCI| image:: https://api.travis-ci.org/tuxcanfly/merchant.png?branch=master
.. _TravisCI: https://travis-ci.org/tuxcanfly/merchant

Django-Merchant is a django application that enables you to use
multiple payment processors from a single API.

Gateways
---------

Following gateways are supported:

* Authorize.net
* Paypal
* Eway
* Braintree Payments (Server to Server)
* Stripe
* Paylane
* WePay
* Beanstream
* Chargebee

Off-Site Processing
--------------------

* Paypal
* RBS WorldPay
* Google Checkout
* Amazon FPS
* Braintree Payments (Transparent Redirect)
* Stripe.js
* eWAY
* Authorize.net Direct Post Method

Other
-----

* Bitcoin
